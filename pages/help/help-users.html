UserSearch

The user search section is a section created for looking for githubUsers, and being able to check what their profiles says about them, and find different information. As you can see, it looks like this:

The most remarkable part of this page is the search input box, which is used to search users, using the right github syntax. If you have any doubts, you can refer here to read github's username guidelines

Once you're done with the input, you can either press enter, or click on the search button for the system to start searching, you will then see the loader showing in the middle of the screen, while the results load, it doesn't take long

When the system is done, you will find the results of the search below

You'll see basic user information about his profile on every card:

If you click on the item, a pop up will open, showing user's profile info, and two buttons to interact with

The first button will take us to the user's github profile, while the second button will take us to another page, to see the user's repositories, this will not go directly to github page, instead, will show a list of the repositories the user has

If you click the first button, as said, you'll see the user's profile

As for the second button, it will take us to this page, where we'll see a list of the user's repositories, and some information about them

We'll also have the chance to click on this button, which will lead us to the user's repository page

Finally, you have the buttons for going to the next, or previous page
