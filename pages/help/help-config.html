Configurations

You can access your user's configurations from the dropdown with your username. This is the configurations view:

Here, apart from the fields to modify the user's information, we will have two buttons, one is for discarding the changes, and the other one is for saving them. If you try to set the exact same user data, an error will display saying that the user already exists.

Besides, in case you have done the changes properly, you will receive a successful message after clicking the save button.

At last, you will have the logout button at the left side of the page. From which you will be able to logout.
